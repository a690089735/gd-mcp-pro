# 产品上下文

## 项目存在的原因
上游 Godot MCP Pro 的 MCP Server 是付费的 Node.js/TypeScript 实现。本 fork 提供一个**免费、开源的 Python 替代品**，让开发者无需购买即可让 AI 助手驱动 Godot 编辑器。

## 解决的问题
1. **AI 无法直接操控 Godot**：传统方式下，AI 只能生成代码片段让用户手动粘贴。MCP 协议让 AI 可以直接创建场景、编辑节点、运行游戏、截图验证。
2. **付费门槛**：上游 Node.js server 需要一次性购买（$15），本 fork 去除了这个门槛。
3. **Python 生态兼容**：许多 AI 工具链基于 Python（FastMCP 等），Python server 更容易集成和扩展。

## 目标用户
- 使用 Cline、Claude Code、Cursor 等 AI 编程助手的 Godot 4.x 游戏开发者
- 希望用 AI 加速场景搭建、脚本编写、调试的独立开发者
- 对 MCP 协议感兴趣的技术探索者

## 典型使用场景
1. **AI 搭建场景**：「帮我创建一个 2D 平台跳跃关卡，包含地面、平台和玩家」
2. **AI 编辑脚本**：「给 Player 节点的脚本添加跳跃功能」
3. **AI 调试运行**：「运行游戏，截图看看当前画面，按下空格键测试跳跃」
4. **AI 批量操作**：「把所有 Sprite2D 的 modulate 改为红色」
5. **AI 检查状态**：「获取场景树结构，看看有没有多余的节点」

## 用户体验目标
- **零门槛启动**：`pip install -e server` + 在 Godot 启用插件 → 即可使用
- **连接可靠**：心跳保活（10s ping）+ 端口自动避让（6505-6514）
- **全功能覆盖**：172+ 工具覆盖场景、节点、脚本、动画、物理、粒子、音频、导航、TileMap、3D、测试等全部领域
- **与上游同步**：新版本 addon 更新不影响 Python server 兼容性
